# Baiteroids

A spin on the classic Asteroids arcade game! Keep yourself safe by smashing
asteroids together as they gravitate towards you!

## Running

The fastest way to get everything you need is to use Nix and Nix flakes.

Once you have Nix with flakes enabled, the development environment with all
necessary dependencies can be entered:

```
nix develop
```

To build and run the game, run:

```
zig build run
```
